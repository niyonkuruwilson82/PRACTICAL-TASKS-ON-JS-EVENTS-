**Form Events Practice Project**
**Description**

This project is a simple webpage that demonstrates how to use HTML Form Events with JavaScript.
It helps students understand how web pages respond to user actions.

** Features**
Input fields for Name and Email
Submit button → shows alert when form is submitted
Reset button → clears the form
Change event → logs input value in the console
Blur event → shows message when input loses focus
 Technologies Used
HTML
CSS
JavaScript
b
Open the project folder
Double-click the HTML file
The page will open in your browser
**Learning Objectives**

Students will learn:

How to use onsubmit event
How to use onchange event
How to use onblur event
How JavaScript interacts with HTML forms
** Example Usage**
Type your name → check console for changes
Click outside input → see alert message
Click submit → form submitted alert appears
